# ML-based Biomarker Discovery for Wilson Disease

## Overview
This repository contains a transcriptomics and machine learning pipeline to identify candidate biomarkers associated with Wilson disease using public gene expression data.

2. Differential gene expression analysis

Identified biomarkers are candidates and require experimental validation.
